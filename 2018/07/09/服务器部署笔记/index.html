nginx

413 Request Entity Too Large

部署好服务，上传图片时，发现nginx报错413 Request Entity Too Large，报错原因是nginx不允许上传配置过大的文件，因此适当调整配置文件即可。

  • 配置路径： /etc/nginx/nginx.conf
  • 修改方法：在http部分，增加client_max_body_size 4m;配置，其中4m表示允许最大上传为4M大小
  • 重启： nginx restart
